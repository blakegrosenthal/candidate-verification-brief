# Candidate Verification Brief

A standalone React + Tailwind prototype for a recruiting prep layer. The demo focuses on helping recruiters see what to trust, what to question, and what to verify before the first interview.

## What is included

- Dashboard summary for one Commercial Account Executive role
- Role criteria panel with must-haves, nice-to-haves, dealbreakers, location, compensation, and screening instructions
- 12 realistic sample candidates with varied fit and verification risk patterns
- Interactive candidate table with search and filters
- Candidate Verification Brief detail panel with credible evidence, claims to verify, gaps, generic answer notes, instruction following, and first-screen questions

## Run locally

```bash
npm install
npm run dev
```

Open `http://127.0.0.1:5173/`.
